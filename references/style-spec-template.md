## 配色规范

### 主色

主色决定整体气质，通常用于：

- 大面积背景；
- 标题区；
- 核心形状；
- 主视觉色调；
- 关键模块底色。

### 辅色

辅色用于承接层级：

- 卡片背景；
- 次级分区；
- 弱信息；
- 表格浅底；
- 辅助图形。

### 点缀色

点缀色只能小面积强调：

- 数据高亮；
- 按钮；
- 重点文字；
- 小徽章；
- 关键箭头或节点。

点缀色大面积使用会喧宾夺主。检查时必须问：这页最亮的颜色是否真的落在最重要的信息上？

### 背景色

背景色决定页面呼吸感：

- 浅底适合密集信息和商务汇报。
- 深底适合封面、章节页、科技感和视觉冲击。
- 纹理 / 渐变背景必须不影响文字可读。

## 字体规范

全套 PPT 建议标题和正文各 1 款字体，最多 2 款字体系统。检查：

- 标题字体是否统一？
- 正文字体是否统一？
- 字重是否有规则，例如标题 Semibold、正文 Regular？
- 是否出现每页不同字体？
- 是否出现中文和英文字体风格冲突？

输出字段：

```text
标题字体：
正文字体：
数字字体：
标题字重：
正文字重：
字号层级：
禁止字体：
```

## 形状规范

形状语言必须统一：

- 圆角系统：所有卡片、按钮、标签统一圆角半径。
- 直角系统：适合严肃商务、科技、工程。
- 缺角 / 异形：适合中国风、科技风或特殊主题，但必须全套一致。
- 线条粗细：边框、连接线、图标线宽要统一。
- 阴影：阴影方向、模糊、透明度不能每页乱变。

输出字段：

```text
形状类型：
cornerRadius：
线条粗细：
边框颜色：
阴影方向：
阴影透明度：
禁止混用：
```

## 图标规范

图标检查维度：

- 线性 / 面性是否统一；
- 单色 / 多色是否统一；
- 线宽是否统一；
- 圆角端点 / 直角端点是否统一；
- 图标语义是否与文字对应；
- 单色图标需要多色时，是否可取消组合后重填色。

不要混用线性和面性图标。不要把不同网站下载的图标直接放在同一页。

## 图片规范

图片决定气质。先定图片类型：

- 实物摄影；
- 扁平插画；
- 3D 渲染；
- 人物照片；
- 场景图；
- 抽象材质；
- 水墨 / 纸张 / 纹理。

统一检查：

- 色调是否一致？
- 饱和度是否一致？
- 亮度是否一致？
- 色温是否一致？
- 裁剪比例是否一致？
- 是否有一张图突兀地改变气质？

找不齐同类素材时，用 PPT 图片格式里的饱和度、亮度、色温拉齐。

## 背景规范

背景类型必须统一：

- 纯色；
- 浅纹理；
- 线性渐变；
- 深色光感；
- 纸张材质；
- 水墨；
- 城市 / 科技场景。

背景应服务内容，不应抢主信息。复杂背景需要用半透明遮罩、渐变矩形或模糊降低干扰。

## 行业配色语境

| 语境 | 推荐倾向 | 谨慎使用 |
|---|---|---|
| 医疗 / 健康 | 蓝绿、浅蓝、干净低饱和、专业感 | 大面积黑白、血红、脏灰 |
| 金融 / 商务 | 深蓝、金色、灰白、低饱和稳重色 | 大面积粉色、高饱和糖果色 |
| 科技 / 城市 | 深蓝、青蓝、紫蓝、光感渐变 | 无意义霓虹、过多发光线条 |
| 教育 / 培训 | 清爽蓝绿、暖黄点缀、亲和浅色 | 过暗背景、压迫感强的黑红 |
| 政务 / 公共 | 红蓝、深蓝、白底、克制金色 | 娱乐化、二次元、过度潮流 |

配色必须同时看行业和情绪。不能因为流行就直接套用。

## 清爽风示例

Design Spec：

```text
风格名称：清爽风 / 清爽科技风
主色：低饱和蓝或蓝紫
辅色：浅蓝、浅灰、白
点缀色：少量青色或黄绿色
背景色：白色或极浅冷色
标题字体：清晰无衬线
正文字体：无衬线 Regular
形状圆角：中小圆角
图标类型：线性或单色面性，保持一致
图片类型：浅色、干净、低饱和
背景材质：纯色、轻渐变、浅纹理
点缀色允许用途：重点数字、按钮、小徽章
禁止混用：厚重纹理、高饱和大面积、复杂写实背景
```

## 中国风示例

Design Spec：

```text
风格名称：中国风 / 新中式
主色：古朴黄、米色、宣纸色、深墨色
辅色：灰褐、浅金、淡红
点缀色：朱砂红或高对比金色
背景色：宣纸、浅纹理、水墨淡底
标题字体：书法感或宋体气质字体
正文字体：可读性强的宋体 / 黑体
形状圆角：缺角、折角或直角系统
图标类型：单色线性或篆刻式小章
图片类型：水墨、古建、器物、纹样
背景材质：纸张、水墨、淡纹理
点缀色允许用途：印章、重点字、小标签
禁止混用：现代霓虹、大面积糖果色、卡通图标
```

## 禁区

- 套模板时不提取原模板 3-4 色，直接手动换色。
- 点缀色大面积铺满页面。
- 同一套 PPT 混用线性图标和面性图标。
- 同一套 PPT 同时混用实物图、扁平插画、二次元和 3D 立体图。
- 每页更换字体、圆角、阴影和背景材质。
- 背景图好看但让文字不可读。
- 只说“科技风 / 高级风”，不写 Design Spec。
