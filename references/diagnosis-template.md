# PPT 页面审美诊断模板

## 输入判断

先判断用户给的是什么：

- 一页 PPT 截图；
- 多页截图；
- 参考图；
- 原始内容；
- 素材图；
- 已有 PPT 文件；
- 视频课程或教程录屏。

如果只是原始内容且没有页面层级，先路由到 `PPT 文案结构化流程`。内容未结构化，不进入审美设计。

## 3 秒判断

3 秒判断不是审美形容词，而是阅读偏差检查：

```text
第一眼看到什么：
应该先看到什么：
最大偏差是什么：
读者会不会知道这页要讲什么：
```

常见偏差：

- 看到装饰图，而不是结论。
- 看到一堆文字，而不是主信息。
- 看到多个颜色同时抢焦点。
- 看到素材，但不知道它和内容有什么关系。
- 看到图表，但看不出数据关系。

## 前置结构门槛

审美前检查：

- 页面任务：这页要证明、说明、对比、总结还是引导？
- 页面标题：是否是结论型标题，而不是分类名？
- 主信息：是否 3 秒内可见？
- 层级：标题、一级要点、二级说明是否分开？
- 信息关系：并列、递进、因果、对比、流程、系统构成是否明确？
- 文案密度：页面是不是像文章？

不通过时，输出：

```text
前置结构状态：不通过。
原因：
下一步：先使用 PPT 文案结构化流程完成结构化检查，再进入小木审美。
```

## 审美金字塔判断

```text
当前层级：A 纯色 / B 渐变 / C 蒙版透明 / D 创意组合
是否跳级：
是否为了升级牺牲可读性：
下一步只升级一个变量：
禁止同时新增的效果：
```

规则：

- A 层问题未解决时，不要强上 C / D。
- 如果文字层级和版式骨架不清，任何渐变、蒙版、光效都只是掩盖问题。
- 升级只改一个变量，例如先从纯色卡片改为轻渐变，不要同时加光效、纹理、复杂背景和多图拼贴。

## P0-P5 问题优先级

| 优先级 | 问题 | 判断 |
|---|---|---|
| P0 内容结构错误 | 页面任务、主信息、信息关系不清 | 先回文案梳理 |
| P1 版式骨架错误 | 重心乱、分区乱、阅读路径乱 | 先画 wireframe |
| P2 风格统一错误 | 字体、形状、图标、图片、背景不统一 | 先提取 Design Spec |
| P3 素材准入错误 | 素材不辅助信息、不提升美感或抢主体 | 保留 / 压低 / 改造 / 删除 |
| P4 样式表现不足 | 元素太默认、缺少层级和节奏 | 加底衬、标签化、图标化、渐变等 |
| P5 操作细节粗糙 | 拉伸、边缘硬、对齐不准、透明度不稳 | 用 PPT 原生操作修细节 |

诊断必须先排序，不要平均用力。

## 四维诊断

### 样式

检查：

- 标题、数字、标签、图标是否过于默认？
- 是否有非默认但克制的处理？
- 样式是否服务层级，而不是纯装饰？
- 是否同一页面里样式太多？

输出动作：

- 标题加底衬 / 色块 / 标签。
- 数字放大并明确单位。
- 并列要点图标化。
- 弱信息降低对比度。

### 版式

检查：

- 主体分区是否清楚？
- 视觉重心是否稳定？
- 留白是否被塞满？
- 信息组之间的距离是否体现关系？
- 是否需要先画 wireframe？

输出动作：

- 用灰色色块重画骨架。
- 三栏 / 左右分栏 / 卡片组 / 流程 / 中心辐射重排。
- 拉开主信息和辅助信息距离。
- 用统一边距和对齐线修复秩序。

### 配色

检查：

- 主色、辅色、点缀色是否明确？
- 点缀色是否失控？
- 是否符合行业和情绪语境？
- 是否有模板原色被随意替换？

输出动作：

- 提取 3-4 色 Design Spec。
- 统一全页文字、形状和图表色。
- 点缀色只用于数据高亮 / 按钮 / 重点文字。
- 删除多余高饱和色。

### 素材

检查：

- 素材是否辅助信息？
- 是否提升美感？
- 是否造成拥挤、杂乱、失焦？
- 是否可编辑，或可用 PPT 原生对象替代？
- 图片色调、亮度、饱和度是否一致？

输出动作：

- 无效素材删除。
- 抢主体素材压低透明度并放底层。
- 图片尺寸不足用渐变矩形延展。
- 多图用柔化边缘、透明度、图层叠加融合。
- 图表根据数据关系替换。

## 修改动作

输出修改建议时按顺序：

1. 先改结构：标题、信息层级、主关系。
2. 再改版式：wireframe、分区、重心、阅读路径。
3. 再定风格：Design Spec、字体、形状、图标、图片、背景。
4. 再处理素材：准入、压低、改造、删除。
5. 最后修细节：对齐、边距、透明度、柔化、图层。

## PPT 原生操作

建议必须尽量落到 PPT 原生功能：

- 形状：矩形、圆角矩形、缺角形、线条、连接器。
- 填充：纯色、线性渐变、透明度、图片填充。
- 图片：裁剪、柔化边缘、透明度、饱和度、亮度、色温。
- 图标：取消组合、重新填色、统一线性 / 面性。
- 图表：条形图、柱状图、环形图 / 圆环图、标签外置。
- 图层：置于底层、叠加、蒙版、遮罩。

不要只说“换个高级背景”。

## 复查标准

- 3 秒内能否看见主信息？
- 页面标题是不是结论？
- 样式、版式、配色、素材是否一致？
- 点缀色是否只承担强调任务？
- 素材是否服务信息或提升美感？
- 数据图表是否表达了正确关系？
- 关键文字、数字、图表是否可编辑？
- 页面是否避免了尺寸硬拉、边缘生硬和图层脏乱？

## 可选评分

仅在用户要求打分时输出：

```text
结构清晰度：/10
版式骨架：/10
风格统一：/10
素材准入：/10
PPT 原生可维护性：/10
当前审美金字塔层级：
最该先改的一项：
```
